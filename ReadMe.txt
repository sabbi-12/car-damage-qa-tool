# Car Damage Detection Web App

A simple **Car Damage Detection** application built using **YOLOv8** and **Streamlit**. This project detects damaged parts on cars from images and visualizes the results with bounding boxes. Ideal for learning **computer vision**, **object detection**, and **web app deployment** with Python.
